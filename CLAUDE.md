# nero-mem2

AI 대화를 위한 로컬 메모리 인프라.

## Tech Stack
- TypeScript (ESM)
- SQLite (better-sqlite3)
- Hono (HTTP server)
- Vitest (testing)
- HuggingFace Transformers (embeddings)

## Commands
- `npm run dev` — 서버 + 웹 UI 동시 실행
- `npm run dev:server` — 백엔드만
- `npm run dev:web` — 웹 UI만
- `npm run build` — TypeScript 빌드
- `npm test` — 테스트 실행
