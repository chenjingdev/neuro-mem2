사용자는 neuro-mem2를 단순 RAG가 아닌 **자아를 가진 자율 에이전트**의 메모리 인프라로 만들고 있다.

핵심 신념:

- 에이전트는 자기만의 성격, 판단 원칙, 목소리를 가져야 한다
- 동시에 소통하는 인간의 Identity도 깊이 이해해야 한다
- 에이전트의 초기 자아는 사용자 분석에서 시드 → 경험으로 진화 (씨앗 + 진화)
- 기존 AI 대화 데이터를 모아놨고, 이를 부트스트래핑에 사용할 예정
- 자율 행동도 원함: 선제 질문, 독립 판단, 메타인지 기반 솔직함
